# ルール整理メモ

## 1. このゲームでやりたかったこと

5x5 という小さい盤面でも、

- 不確定な位置取り
- 観測での確定
- 観測するか、放置して圧をかけるか

という判断が出るようにしたかった。

ただし、量子っぽさを優先しすぎると対局ルールが曖昧になりやすい。特に「量子捕獲」「王が死んでいる世界と生きている世界が同時にある」状態まで許すと、UI も AI も一気に重くなる。そこで今回は、遊べることと説明できることを優先して整理した。

## 2. 採用した整理

### 量子移動

- 確定駒だけが量子移動できる
- 移動先は 2 マス
- どちらも非捕獲マスである必要がある
- 分岐比率は 50% / 50%

### 観測

- 不確定駒に対して行う
- 観測すると状態全体が 1 世界に collapse する
- 観測も 1 手として数える

### 捕獲

- 古典移動のみ
- 対象は確定している相手駒のみ

## 3. なぜ world mixture を使ったか

駒ごとに「このマス 30%、あのマス 70%」と直接持つ方法もあるが、それだと複数駒の相関が崩れる。たとえば

- A が左にいる世界では B は右にいる
- A が右にいる世界では B は中央にいる

というような関係を表しづらい。

そこで、局面全体を確率付きの複数世界として持つことにした。こうすると

- 盤面描画では周辺確率を集計すればよい
- AI では期待値計算にそのまま使える
- リプレイでも局面をそのまま保存できる

という利点がある。

## 4. 合法手の考え方

不確定状態があると、「ある世界では通るが、別の世界では通らない」手が出る。これを許すと、プレイヤーが何を指したのか自体が曖昧になる。

そこで今回は、合法手を

> すべての残存世界で同じ意味を持つ手

に限定した。

これにより、

- スライド駒の経路に不確定駒がいるなら、その筋の手は通らない
- 捕獲対象が世界ごとに変わる手は通らない
- まず観測して盤面を固める価値が生まれる

という整理になる。

## 5. 本将棋との違い

- 持ち駒なし
- 成りなし
- 王手・詰みよりも「確定捕獲」を勝敗条件にした
- 千日手や入玉のような細則は未実装

5x5 の短いゲームとして回しやすくするための簡略化であり、ここを広げるときは別バリアントとして切り分ける想定。
